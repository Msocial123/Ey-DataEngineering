Retail Datasets for Data Engineering Project (Larger samples)

Files in this folder:
- products.parquet or products.csv (product master)
- customers.json (customer master)
- inventory.csv (inventory per product)
- sales.csv (order-level sales)
- payments.csv (payment transactions)
- returns.csv (returns)

Use these files to demonstrate reading/writing CSV, JSON, Parquet, and to build Delta tables in Databricks.
